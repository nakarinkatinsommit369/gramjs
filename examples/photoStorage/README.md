# แอพเก็บภาพ (ตัวอย่าง)

ตัวอย่างนี้เป็นแอพเก็บภาพแบบง่าย ใช้ Node.js HTTP server สำหรับรับภาพและแสดงแกลเลอรีแบบหน้าเว็บเดียว

## วิธีรัน

```bash
node examples/photoStorage/server.js
```

จากนั้นเปิดเบราว์เซอร์ไปที่ `http://localhost:3000`

## วิธีใช้งาน

1. ใส่ชื่อภาพ (หรือปล่อยว่างเพื่อใช้ชื่อไฟล์เดิม)
2. เลือกไฟล์ภาพจากเครื่อง
3. กดปุ่ม "บันทึกภาพ" แล้วภาพจะถูกเพิ่มลงในแกลเลอรีด้านล่าง

## โครงสร้าง

- `public/index.html` หน้าเว็บสำหรับอัปโหลดและดูภาพ
- `uploads/` โฟลเดอร์เก็บภาพที่อัปโหลด
- `server.js` เซิร์ฟเวอร์ API สำหรับบันทึกและอ่านรายการภาพ
